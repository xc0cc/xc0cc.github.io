C#基本语法

C# 入门：从零开始的基本功

一个 C# 程序长什么样

逐层拆开：

  • using System——引入系统基础工具包。System 是顶级命名空间，在 .NET 6+ 中默认已隐式引入，可以省略
  • namespace——命名空间，组织代码、避免命名冲突。可理解为”这个文件属于哪个文件夹”
  • class——类是 C# 组织代码的基本单位，所有逻辑必须放在类里
  • static void Main——程序入口，运行时从这里开始。每个程序只有一个 Main
  • Console.WriteLine——控制台打印一行，自动换行

#region / #endregion 用来折叠代码块，纯粹为了可读性，不影响运行。

变量和数据类型

声明与赋值

规则：先声明再使用、不能重名、不能数字开头、不能用关键字当变量名。

数值类型速查

写代码时 90% 的情况只需要这几个：

类型 用途 示例
int 整数，最常用 int a = 10;
long 大整数 long l = 9999999999L;
float 单精度浮点 float f = 0.1f;
double 双精度浮点，小数默认类型 double d = 0.1;
decimal 高精度，适合财务计算 decimal m = 0.1m;

float 必须加 f 后缀，decimal 必须加 m 后缀。不带后缀的小数默认识别为 double。

sbyte、ushort、uint、ulong 等类型知道有就行，日常极少用到。

特殊类型

类型 说明 示例
bool true 或 false bool ok = true;
char 单个 Unicode 字符，占 2 字节 char c = 'A';
string Unicode 字符串，无长度上限 string s = "你好";

命名习惯

  • 变量/参数：驼峰 myName、maxHealth

常量

声明时必须赋值，之后不可更改。适合数学常数、配置上限这类不变的值。
