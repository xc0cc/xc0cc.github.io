是什么和能做什么

是什么：

Python是一种高级编程语言，具有简洁、易读、易维护等特点，被广泛应用于科学计算、人工智能、数据分析、Web开发、自动化测试等领域。Python语言简洁易读的语法、丰富的第三方库、跨平台的特性以及大量优秀的框架等，使得Python语言备受开发者青睐，成为目前最流行的编程语言之一。另外，Python还具有开源、免费、社区活跃等特点，使得更多的开发者加入到Python社区中，使Python在不断发展壮大的同时，也形成了强大的生态系统。

能做什么：

  • 爬虫
  • 科学计算
  • 数据处理到数据分析
  • 人工智能、搭建神经网络
  • Web开发
  • 自动化测试、自动化运维

非常多，越来越多人去使用和构建社区，有非常强大的生态系统。

代码规范

挺重要，因为python会根据缩减判断，也会对大小敏感。

  1. 文件名和文件路径不要出现中文 禁止使用空格

  2. 文件名不要不要如系统的包重复，会报错，如numpy.py（md）

  3. 标识符：

    1. 标识符由字母+数字+下划线组成
    2. 不能以数字开头
    3. 区分大小写
    4. 单下划线开头，代表不能直接访问的类属性
    5. 双下划线开头，代表特殊方法
    6. 一行多条语句用分号;分开

  4. 保留字：

    1. 所有的Python关键字只包含小写字母
    2. and/break/continue/import/def

  5. 行和缩进：

    1. 缩减表明模块，严格执行缩减
    2. 一般新行作为语句/模块的结束符
    3. 把一个Tab改成四个缩进

  6. 多行语句：

    1. 使用斜杠\将一行语句分为多行显示
    2. 语句包含[],{}.()就不需要多行连接符

  7. 引号：

    1. 引号开始和结尾必须使用想用类型（””.’’.”””）
    2. 三引号组成多行，用于文档注释

  8. 注释：

    1. 单行注释用#开头

  9. 空行

    1. 函数之间或类之间用空行分割，表示一段新代码的开始
    2. 类和函数入口之间也可用一行空行分割，以突出函数入口的开始

  10. 类名

    1. 大驼峰命名法

    2. CapWords

    3. 每个单词首字母大写

    4. 单词和单词之间没有下划线

  11. 导入模块时，每个导入应该独占一行

  12. 导入顺序

    1. 内置模块
    2. 自定义模块
    3. 第三方模块
